CONTROLES:

JUEGO
A -> girar línea de apuntado a la izquierda
D -> girar línea de apuntado a la derecha
K -> Ajustar potencia -> Darle otra vez -> Tirar
R -> restear el proyectil
B -> cambiar tipo de proyectil (ligero/estándar)
T -> cambiar tipo de tiro (fuerte/colocado)
M -> activar/desactivar modo Volea (solo con pelota de fútbol).
FUERZAS
V -> activar/desactivar viento
